# 💪 CRUD Academia

Sistema web com autenticação de usuários e gerenciamento de dados.

## 🚀 Funcionalidades
- Login de usuários
- Cadastro de registros
- Edição e exclusão de dados
- Integração com banco de dados

## 🛠️ Tecnologias
- Java / PHP (ajusta aqui)
- MySQL
- HTML / CSS / JavaScript
